Accessible Tariff Switcher Component

Requirements

  • Variable number of tariffs
  • Only one tariff can be selected at one time
  • Selecting a tariff is mandatory
  • Accessible and selectable via keyboard
  • Screenreader should properly read the options and selection
  • An event should be dispatched when a tariff is selected
  • The component should have an API with methods to:
    • Get the selected tariff
    • Get all the available tariffs
    • Modify the selection programmatically
